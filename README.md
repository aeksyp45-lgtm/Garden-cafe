# ຮ້ານກາເຟ ບ້ານສວນ — ລະບົບ POS

ໂປຣເຈັກນີ້ແມ່ນລະບົບຂາຍໜ້າຮ້ານ (POS) ສຳລັບຮ້ານກາເຟ, ສ້າງດ້ວຍ React.
ຂໍ້ມູນການຂາຍຈະຖືກບັນທຶກໄວ້ໃນ browser ຂອງເຄື່ອງທີ່ໃຊ້ (localStorage).

## ວິທີໃຊ້ຢູ່ໃນຄອມພິວເຕີຂອງທ່ານ (ທົດລອງກ່ອນ)

ຕ້ອງຕິດຕັ້ງ Node.js ກ່ອນ (ດາວໂຫລດຟຣີໄດ້ທີ່ https://nodejs.org — ເລືອກເວີຊັນ LTS)

ຫຼັງຈາກນັ້ນ ເປີດ Terminal (ຫຼື Command Prompt) ໃນໂຟນເດີນີ້ ແລ້ວພິມ:

```
npm install
npm run dev
```

ຈາກນັ້ນເປີດ browser ໄປທີ່ລິ້ງທີ່ຂຶ້ນມາ (ປົກກະຕິແມ່ນ http://localhost:5173)

## ວິທີເອົາໄປໃຊ້ຈິງທີ່ຮ້ານ (Deploy)

### ຂັ້ນທີ 1 — ອັບໂຄ້ດຂຶ້ນ GitHub
1. ສ້າງບັນຊີຟຣີທີ່ https://github.com
2. ສ້າງ repository ໃໝ່ ແລ້ວອັບໂຟນເດີນີ້ຂຶ້ນໄປ (ໃຊ້ GitHub Desktop ຖ້າຍັງບໍ່ຄຸ້ນກັບ git command)

### ຂັ້ນທີ 2 — Deploy ຜ່ານ Vercel (ຟຣີ)
1. ສະໝັກບັນຊີທີ່ https://vercel.com ໂດຍໃຊ້ບັນຊີ GitHub ດຽວກັນ
2. ກົດ "New Project" ແລ້ວເລືອກ repository ທີ່ອັບໄວ້
3. ກົດ Deploy — Vercel ຈະສ້າງລິ້ງເວັບໄຊທ໌ໃຫ້ອັດຕະໂນມັດ (ເຊັ່ນ `cafe-pos-xxxx.vercel.app`)

### ຂັ້ນທີ 3 — ໃຊ້ຢູ່ຮ້ານ
1. ເປີດລິ້ງນັ້ນຢູ່ tablet ຫຼືຄອມພິວເຕີຢູ່ຮ້ານ
2. ໃນ browser ມືຖື/tablet, ເລືອກ "Add to Home Screen" ເພື່ອໃຫ້ມີໄອຄອນຄືແອັບ

## ຂໍ້ຄວນຮູ້ (ສຳຄັນ)

- **ຂໍ້ມູນຜູກກັບ browser/ເຄື່ອງ**: ຖ້າໃຊ້ຫຼາຍເຄື່ອງ (ຫຼາຍ tablet), ແຕ່ລະເຄື່ອງຈະມີປະຫວັດການຂາຍແຍກກັນ, ບໍ່ໄດ້ sync ກັນ. ຖ້າຢາກໃຫ້ຂໍ້ມູນລວມສູນ ຕ້ອງໃຊ້ຖານຂໍ້ມູນຄລາວ (ເຊັ່ນ Supabase) ແທນ localStorage — ຂ້ອຍຊ່ວຍປັບໃຫ້ໄດ້ຖ້າຕ້ອງການ
- **ຖ້າລ້າງ browser cache**: ຂໍ້ມູນປະຫວັດການຂາຍຈະຫາຍໄປ, ຄວນສຳຮອງຂໍ້ມູນເປັນປະຈຳ
- **ໂຄງສ້າງໄຟລ໌**: ແກ້ໄຂເມນູ/ລາຄາໄດ້ທີ່ `src/App.jsx` ໃນສ່ວນ `CATEGORIES`
